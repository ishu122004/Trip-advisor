# TravelExplorer

A responsive travel website built using HTML and CSS that helps users discover destinations, hotels, travel experiences, and airline partners through an attractive and user-friendly interface.

## Features

* Responsive navigation bar
* Search functionality UI
* Travel destination showcase
* Hotel discovery section
* Travel experience cards
* Airline partner section
* Interactive animations and hover effects
* Mobile-friendly responsive design

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
* Font Awesome
* Git
* GitHub
